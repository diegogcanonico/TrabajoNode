1. Clonar el repositorio con 
```git clone https://github.com/nexun/curso-node.git```
    
2. Moverse por consola a la carpeta donde se encuentra nuestro archivo package.json
```cd curso-node/```
3. Ejecutar 
```npm install```
4. Recuerden instalar nodemon con el comando
``` npm i -g nodemon```
5. Finalmente 
 ```npm start```
